TBA3241 SOCIAL NETWORK ANALYSIS – L1 TO L12 SUMMARY

----------------------------------------
L1 – NETWORK BASICS
----------------------------------------
- Nodes, edges
- Degree, path, shortest path
- Eccentricity = max distance from node
- Diameter = longest shortest path
- Density = edges / possible edges

----------------------------------------
L2 – NETWORK METRICS
----------------------------------------
- Density, connectivity
- Average path length
- Small-world networks

----------------------------------------
L3 – CENTRALITY
----------------------------------------
- Degree → popularity
- Closeness → speed
- Betweenness → bridge/control
- Eigenvector → influence via important nodes

----------------------------------------
L4 – SUBNETWORKS
----------------------------------------
- Clique (fully connected)
- Maximal vs Maximum clique
- n-clique, n-clan, n-club

----------------------------------------
L5 – COHESION
----------------------------------------
- k-core → degree ≥ k
- k-plex → allow missing links
- Clustering coefficient
- Components, fragmentation

----------------------------------------
L6 – TIES
----------------------------------------
- Strong vs weak ties
- Reciprocity
- Homophily
- Triadic closure

----------------------------------------
L7 – STRUCTURAL BALANCE
----------------------------------------
- + (friend), – (enemy)
- Balanced:
  - +++ OR +--
- Leads to 2 groups

----------------------------------------
L8 – INFORMATION FLOW
----------------------------------------
- Flow network
- Max-flow min-cut
- Ford-Fulkerson
- Bottleneck = key control

----------------------------------------
L9 – NETWORK DYNAMICS
----------------------------------------
- Cascade model
- Threshold: p ≥ b / (a + b)
- Clusters stop cascades

----------------------------------------
L10 – NETWORK EFFECTS
----------------------------------------
- SIR / SIS models
- R₀ = p × k
- Network externality

----------------------------------------
L11 – PLATFORM GOVERNANCE
----------------------------------------
- Content moderation
- Misinformation
- Regulations (UK, EU, SG)

L12 – COURSE INTEGRATION
